##  Tech Stack

### Frontend
* **React:** Component-based UI library.
* **Vite:** Fast build tool and development server.
* **React Router:** Client-side routing.
* **CSS3:** Custom styling with a monochrome design system.

### Backend
* **Node.js:** Runtime environment.
* **Express:** Web application framework.

### Deployment
* **Vercel:** Cloud platform for hosting.

##  Local Setup

Follow these steps to run the project locally.

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/thinkmag.git](https://github.com/YOUR_USERNAME/thinkmag.git)
    cd thinkmag
    ```

2.  **Install & Run Client (Frontend)**
    ```bash
    cd client
    npm install
    npm run dev
    ```

3.  **Install & Run Server (Backend)**
    ```bash
    cd ../server
    npm install
    npm start
    ```
